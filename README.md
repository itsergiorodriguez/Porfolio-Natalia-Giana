## 📖 Sobre el proyecto

Portfolio personal de Natalia Paola Giana, Licenciada en Trabajo Social con más de 19 años de experiencia en gestión administrativa y coordinación de proyectos. Diseñado para empleadores que buscan perfiles profesionales con sensibilidad humana y disciplina administrativa, especialmente para roles remotos.

## ✨ Características

- **Diseño editorial cálido**: paleta tierra, salvia, durazno y dorado suave
- **Tipografías de calidad**: Cormorant Garamond + Manrope
- **Animaciones SVG sutiles**: red de personas conectadas, decoraciones flotantes con parallax
- **100% responsive**: móvil, tablet y desktop
- **Foto integrada**: embebida en base64 — archivo único, máxima portabilidad
- **Optimizado para reclutadores**: secciones claras de experiencia, formación, competencias y disponibilidad
- **Sin dependencias pesadas**: vanilla HTML/CSS/JavaScript

## 🛠️ Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript (Intersection Observer API)
- SVG inline animado
- Google Fonts (Cormorant Garamond + Manrope)

## 🚀 Cómo correrlo localmente

```bash
git clone https://github.com/itsergiorodriguez/portfolio-natalia-giana.git
cd portfolio-natalia-giana
# Abrir index.html en cualquier navegador moderno
```

No requiere build ni servidor.
